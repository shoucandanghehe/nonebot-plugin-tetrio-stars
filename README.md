TETRIO Stats
===
一个在nonebot2中查询TETRIO玩家数据的插件，同时支持绑定TETRIO账号

安装
---
* 使用 pip
```
pip install nonebot-plugin-tetrio-stats
```
使用
---
参考NoneBot2文档 [加载插件](https://v2.nonebot.dev/docs/tutorial/plugin/load-plugin/)

依赖
---
目前只支持 `OneBot V11` 协议

指令列表
---

 | 指令 | 说明 |
 | --------- | --------- |
 | io查`USERID` | 查询游戏统计信息 |
 | iobind`USERID` | 绑定自己的账号 |
